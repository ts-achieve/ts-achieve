`ts-achieve` is a VS Code extension by T.S. Achieve. It gamifies the TypeScript development experience by organizing diagnostic messages into unlockable achievements.

# development

## setup

```
cd ts-achieve
pnpm i
```

## testing

```
cd ts-achieve
pnpm t
```
